The ``5-text_indentation`` module
=================================

Using ``text_indentation``
--------------------------

First import ``text_indentation`` from ``5-text_indentation`` module:

      >>> text_indentation = __import__('5-text_indentation').text_indentation

Now use it:

    # text is only .
    text_indentation('..')
    .
    <BLANKLINE>
    .
    <BLANKLINE>

    # text is only ?
    text_indentation("????")
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
    ?
    <BLANKLINE>
 
    # text is only :
    text_indentation(":::")
    :
    <BLANKLINE>
    :
    <BLANKLINE>
    :
    <BLANKLINE>

    # text includes :
    text_indentation('grocery list: eggs, bacon')
    grocery list:
    <BLANKLINE>
    eggs, bacon

    # text inludes .
    text_indentation("YOLO.")
    YOLO.
    <BLANKLINE>

    # text includes ?
    text_indentation("Hello?")
    Hello?
    <BLANKLINE>

    # text includes all 3
    text_indentation("This is a piece of text that contains: a ? and .")
    This is a piece of text that contains:
    <BLANKLINE>
    a ?
    <BLANKLINE>
    and .
    <BLANKLINE>
